# Espaço da Gi

PWA pessoal de hábitos, leitura, foco e cuidado, feita para a Gigi.

## Estrutura

- `index.html`: aplicativo completo e responsivo
- `assets/`: imagens locais, sem URLs externas
- `icons/`: ícones Android/iPhone
- `manifest.webmanifest`: instalação como aplicativo
- `sw.js`: funcionamento offline básico

## Publicação

Importe este repositório na Vercel como projeto estático. Não há comando de build.
